### 為何要學CLI

    沒有為什麼。學習不需要理由，你要做的只有聽話照做。如果說真的非得給學習 CLI 強制安上一個
原因，那就是學會之後，你操作起電腦會比別人看起來更帥。當別人滑鼠壞掉，只能急著去光南買一個大
特價 99 塊的滑鼠回來替換的時候，你不疾不徐、抄起鍵盤，用鋼琴家的架式敲打著那些小方塊，叫出一
個黑糊糊的畫面，開始一來一往地跟電腦溝通下指令，靜靜地裝著逼。

    你覺得自己的背影看起來就像基努．李維，旁人握緊手中包裝還沒拆的新滑鼠，努力克制自己不要用
鄙視的神情看向一個第一次摸到電腦的猴子。你朋友不是黑人，但正用所有臉部肌肉全力演繹著黑人問號
表情包，因為你，只知道怎麼打開CMD。

C:\Users>hi
'hi' 不是內部或外部命令、可執行的程式或批次檔。

C:\Users>open browser
'open' 不是內部或外部命令、可執行的程式或批次檔。

    你帥到出水了。從自己的眼眶流了出來。

    你緊握拳頭又緊咬牙關：「幫…幫我把滑鼠拆開…嗚嗚嗚」。

###  為何要學CLI 其二

    當然，以上並不是真正的原因，但是，你仍然需要知道滑鼠只是輔助。你買電腦時並不會像買台新洗
衣機時一樣附上一本密密麻麻的使用說明書，就算有，你也不會看，所以你用了三年多，除了洗衣脫水，
從來沒用過洗衣機的蒸汽殺菌功能，反正衣服都有洗乾淨，你就這樣穿，未曾想過有天會需要再洗得更乾
淨，直到有一天，世界上爆發了一種傳染病，導致只要跟你擦肩的路人清了清喉嚨，你回家就想全面消個
毒，但說明書早就不見了。

    CLI 就像洗衣機的蒸汽殺菌……不，準確來說，就像單眼相機的 M 模式，平常就在那邊待機，但你永
遠只會在自動模式按下快門，所以無法解除展現你真正攝影實力的拘束器。學會了，你才真正懂得什麼叫
拍照。會用 CLI ，普立茲獎就近在眼前。

    「普通人用電腦，都只用了5%。」你今天就來親自實證，告訴大家這句話是假的，因為你可是學會了
CLI ，訓練有素的猴子可是可以用到 5.5% 呢。

    幸虧，電腦這玩意本來就是開發給猴子用的，你不需要靠自己的智慧跟感性來判斷自己變焦環該轉幾
圈、快門要設幾秒，你只要記住訓練者給你的指令，聽話照做，就有香蕉吃，非常 Easy 。

### 指令

    首先，打開 CMD ，或是安裝 Git，再叫出那個黑色的帥氣畫面。
    以下，給你指令跟作用。

| 指令 | 作用 |
| :--- | :--- |
| pwd | 哲學大哉問－我在哪？ |
| ls | 列出檔案清單 |
| cd | 移動至指定位置 |
| man | 叫出指令手冊 |
| touch | 新建檔案 |
| mkdir | 新建資料夾 |
| mv | 移動、改名 |
| cp | 複製 |
| vim 檔名 | 編輯檔案文字 |

    所以，只要在裡面輸入 `mkdir wifi`，就可以建立一個叫做 wifi 的資料夾。
    （為什麼不是 wife？）
    然後，輸入 `cd wifi`，再輸入`touch afu.js`就可以了。

    是不是很簡單？你要告訴電腦你想幹嘛，它就會幫你做到。

    如果想要知道更多用法，去Google吧！有人把這些寶藏都放在那裡了！ㄅㄅ！

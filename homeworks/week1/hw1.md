## 交作業流程

### 簡略版 - In Git Bash -  

0. clone 一份 mentor-program-4th 到本地端 a.k.a 我的電腦

1. 新建 branch 並切換到該 branch
2. vim 指定檔案，開始寫作業
3. 存檔後，將作業加入版本控制（若新增檔案），然後 commit 一個新版本
4. 將新版本 push 到雲端，使用 pull request 請求將自己的作業 merge 到 master
5. 在學習系統裡新增作業
6. 確認批改完成，merge 後將檔案同步到本地端
